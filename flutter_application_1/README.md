# ToDo YOU App

Aplicativo desenvolvido em Flutter para gerenciamento de tarefas.

## Funcionalidades

- Cadastro de usuário
- Login com validação
- Recuperação de senha
- Adicionar tarefas
- Listar tarefas
- Editar tarefas
- Marcar como concluída
- Excluir tarefas
